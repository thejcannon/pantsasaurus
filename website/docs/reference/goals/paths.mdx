```
pants paths [args]
```

List the paths between two addresses. Either address may represent a group of targets, e.g. `--from=src/app/main.py --to=src/library::`.

Backend: <span style={{ color: "var(--ifm-color-primary)" }}>`pants.backend.project_info`</span>

Config section: <span style={{ color: "var(--ifm-color-primary)" }}>`[paths]`</span>
