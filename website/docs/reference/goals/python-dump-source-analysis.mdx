```
pants python-dump-source-analysis [args]
```

Dump source analysis for python_source targets.

Backend: <span style={{ color: "var(--ifm-color-primary)" }}>`pants.backend.experimental.python`</span>

Config section: <span style={{ color: "var(--ifm-color-primary)" }}>`[python-dump-source-analysis]`</span>

## Basic options

<span className={styles.options}>

### `analysis_flavor`

<Option
  cli_repr='--python-dump-source-analysis-analysis-flavor=<AnalysisFlavor>'
  env_repr='PANTS_PYTHON_DUMP_SOURCE_ANALYSIS_ANALYSIS_FLAVOR'
  one_of='raw_dependency_inference, dependency_inference'
  default_repr='dependency_inference'
>
  

The type of information that should be returned.

- `dependency_inference`\: The results of dependency inference, for every detected import in every file.

- `raw_dependency_inference`\: The raw intermediate results of the dependency inference process,
at every stage they're available. Potentially useful for debugging the dependency inference process.

</Option>
