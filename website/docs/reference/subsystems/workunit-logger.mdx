Workunit Logger subsystem. Useful for debugging pants itself.

Backend: <span style="color: purple"><code>pants.backend.experimental.tools.workunit_logger</code></span>
Config section: <span style="color: purple"><code>[workunit-logger]</code></span>

## Basic options

<div style="color: purple">

### `enabled`

  <code>--[no-]workunit-logger-enabled</code><br>
  <code>PANTS_WORKUNIT_LOGGER_ENABLED</code><br>
</div>
<div style="padding-left: 2em;">
<span style="color: green">default: <code>False</code></span>

<br>

Whether to enable workunit logging.

</div>
<br>

<div style="color: purple">

### `logdir`

  <code>--workunit-logger-logdir=&lt;str&gt;</code><br>
  <code>PANTS_WORKUNIT_LOGGER_LOGDIR</code><br>
</div>
<div style="padding-left: 2em;">
<span style="color: green">default: <code>.pants.d</code></span>

<br>

Where to write the log to.

</div>
<br>
