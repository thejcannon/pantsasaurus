See [Targets](reference:reference-all-targets) or run `./pants help targets` for a list of all target types.

Python target types:

- [`python_library`](reference:reference-python_library), Python source code.
- [`python_tests`](reference:reference-python_tests). Also see [test](../goals/test.mdx).
- [`pex_binary`](reference:reference-pex_binary), metadata to create an executable PEX file. Also see [package](../goals/package.mdx) and [run](../goals/run.mdx).
- [`python_distribution`](reference:reference-python_distribution), a publishable Python setuptools distribution (e.g. an sdist or wheel). Also see [Building distributions](./building-distributions.mdx).
- [`python_requirement_library`](reference:reference-python_requirement_library), Python requirements installable by pip. Also see [Third-party dependencies](./third-party-dependencies.mdx).

See [Resources](../../using-pants/resources-and-archives.mdx) for information on the `files`, `resources`, and `archive` target types.
